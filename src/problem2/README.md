# Currency Exchange Application

A modern cryptocurrency exchange application built with React, featuring real-time currency conversion with a clean and intuitive user interface.

## Demo

Visit the live demo: [https://99cryptopcurrency-exchange.netlify.app/](https://99cryptopcurrency-exchange.netlify.app/)

## Requirements

- **Node.js**: Version 20 or higher
- **Package Manager**: yarn (recommended)

## Tech Stack

### Core

- **Vite** - Fast build tool and development server
- **TypeScript** - Type safety and better developer experience

### State Management & Data Fetching

- **React Hook Form** - Efficient form state management
- **TanStack Query (React Query)** - Server state management and data fetching

### Styling

- **Tailwind CSS 4** - Utility-first CSS framework
- **shadcn/ui** - Re-usable component library built with Radix UI

### Utilities

- **Lodash** - Utility functions (debounce for optimized API calls)

## Getting Started

### Installation

```bash
# Install dependencies
yarn install

# Start development server
yarn dev
```
